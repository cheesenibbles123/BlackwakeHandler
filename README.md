Simple npm package to allow easier usage to blackwake commands.

## Install

```text
npm install @cheesenibbles/blackwakehandler
```

## Usage

```js
const bw = require('blackwakehandler');
bw.init(/*Your steamAPI key here*/);
bw.handler(type,steamID64);
```
